What is jQuery?

A fast, concise, library that simplifies how to traverse HTML documents, handle events, perform animations, and server requests via AJAX.

$('#lets').begin().

Conventions in jQuery.

  • $();
    This is the jQuery function, it is short for jQuery().
  • $('.selector');
    jQuery uses selectors, similar to those in CSS for traversing elements in the document.
  • $('.selector').fadeIn(250);
    Calling the fadeIn() method of our jQuery object.
  • $('.selector').click(function() { $(this).fadeOut(250) });
    Bind to the click event, call fadeOut() after the user clicks.
